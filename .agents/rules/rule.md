우리는 지금부터 안티그래비티 환경에서 단 하나의 HTML 파일로 'AI 아카이빙 홈페이지'를 만들 거야. 본격적인 코딩에 앞서 나와 일하는 룰을 정하자.

1. 모든 기능은 한 번에 하나씩 점진적으로 만든다. 내가 오케이 하기 전엔 다음 기능으로 가지 않는다.

2. 코드를 수정할 때 기존에 잘 작동하던 기능이나 스타일이 망가지지 않도록 항상 코드를 더블 체크한다.

3. 라이브러리는 CDN(Tailwind CSS 등)을 활용해 파일 하나로 가볍게 구동되게 한다.

4. 에러가 나면 부끄러워하지 말고 솔직하게 말하고 같이 고친다.

이 룰과 워크플로를 이해했다면, 각오 한마디와 함께 우리가 만들 홈페이지의 가장 기초적인 뼈대(Layout) 코드부터 보여줘!
